# Duino Account Dashboard – Monitorizare live Duino-Coin

**Duino Account Dashboard** este un dashboard web modern, creat pentru monitorizarea în timp real a conturilor și minerilor Duino-Coin. Proiectul combină tehnici moderne de HTML, CSS și JavaScript pentru a oferi o experiență vizuală interactivă și elegantă, fără a compromite performanța.

---

## Caracteristici principale ##

### 1. Interfață intuitivă și organizată
* Structură clară, bazată pe framework-ul **Bulma CSS**, care asigură un layout responsive și ușor de navigat.
* Datele contului (username, balans, date de creare, last login etc.) sunt afișate în carduri individuale, pentru o vizualizare rapidă și coerentă.

### 2. Design vizual modern
* Fundal cu **gradient radial** și efect de **perspectivă**, pentru senzația de profunzime.
* Cardurile și titlurile beneficiază de **animații 3D** și efecte de glow.
* Iconițele și pictogramele sunt animate și aduc dinamism vizual (de exemplu, efectul de rotație a logo-ului).

### 3. Monitorizare live a contului și minerilor
* Interfața preia date în timp real folosind **API-ul Duino-Coin**.
* Actualizările sunt automate, la fiecare 60 de secunde, fără a necesita reîncărcarea paginii.
* Cardurile pentru mineri includ **hashrate individual** și un **total Hashrate** calculat automat.
### De reținut
Pentru monitorizare personală: O interogare la fiecare 30 - 60 de secunde este considerată ideală. Aceasta oferă date actualizate fără a solicita inutil serverele proiectului.
Frecvența maximă sigură: Evitați să faceți mai mult de o solicitare la 5-10 secunde. Solicitările mai dese pot fi interpretate ca un atac de tip spam sau bot. 

### 4. Experiență de utilizator atent gândită
* Valori critice (ex. warnings, status API) sunt evidențiate prin culori și animații pulsante.
* Mesaje informative despre starea serverului și ultimele actualizări ajută utilizatorul să rămână la curent.
* Interacțiuni vizuale plăcute, cum ar fi **hover effects** pe carduri și pictograme.

### 5. Design scalabil și modular
* Codul este ușor de extins: poți adăuga noi informații despre cont sau noi mineri fără a afecta funcționalitatea existentă.
* Clasele CSS și ID-urile sunt bine structurate, ceea ce facilitează personalizarea stilului.

### 6. Elemente interactive și branding
* Include linkuri externe utile (GitHub, Facebook, Explorer, WebMiner) cu pictograme consistente și stilizate.
* Animațiile și culorile respectă o paletă coerentă, oferind un aspect profesional și unitar.
